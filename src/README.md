Pocket Cube Optimal Solver
==========================

Install
-------
```
npm install
```

Develop
-------
```
npm run dev
```

Build for production
--------------------
```
npm run build
```

Publish to GihtHun Pages
------------------------
```
npm run gh-pages
```
This command will build the app and put it to `/docs` folder, which
GitHub uses as the source for GitHub Pages.
